Check Your Sporty Range

Startpunkt festlegen | Sportportaktivitäten Wetterinformationen

Über einen Link wird auf eine eigene HTML-Seite referenziert, welche es dem User ermöglicht, genauere Positionsangaben seiner Startpunkte bzw. seines Startpunkts innerhalb Innsbrucks anzugeben. Dazu findet eine Adressabfrage oder das Setzen eines Markers statt, deren Koordinaten für das weitere Routing herangezogen werden.
